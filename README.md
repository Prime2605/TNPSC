## ğŸ“¦ Installation & Setup

Follow these steps to get the project running on your local machine.

### Prerequisites
- [Node.js](https://nodejs.org/) (v16 or higher)
- [npm](https://www.npmjs.com/) or [yarn](https://yarnpkg.com/)

### Steps

1.  **Clone the Repository**
    ```bash
    git clone https://github.com/yourusername/tnpsc-dashboard.git
    cd tnpsc-dashboard/frontend
    ```

2.  **Install Dependencies**
    ```bash
    npm install
    # or
    yarn install
    ```

3.  **Start the Development Server**
    ```bash
    npm run dev
    # or
    yarn dev
    ```

4.  **Open in Browser**
    Visit `http://localhost:5173` to view the application.

## ğŸŒˆ Design Philosophy

The design of **Subashini** is centered around the concept of **"Motivating Aesthetics"**. We believe that a beautiful tool encourages usage.
